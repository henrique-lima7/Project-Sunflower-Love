##  🌻 Sobre o Universo do Projeto

Este projeto é uma aplicação web front-end de altíssima fidelidade visual e interativa, desenvolvida sob o conceito estético de **girassóis e tons outonais (amarelo, ocre e creme)**. Criado originalmente para celebrar uma história de **5 anos de namoro**, o software transcende uma simples página estática: ele entrega uma jornada de usuário imersiva, combinando design afetivo (*emotional design*) com engenharia de software robusta.

Para **recrutadores e desenvolvedores**, este repositório destaca maturidade técnica em arquitetura de arquivo único limpa, persistência de dados em tempo real, inteligência artificial integrada e acessibilidade ponta a ponta.

## 🛠️  Stack Tecnológica

O projeto foi construído puramente com tecnologias nativas (Vanilla) para garantir máxima performance de carregamento e execução:

* **HTML5 Estrutural & Semântico:** Garantia de SEO básico e acessibilidade estrutural.
* **CSS3 Avançado (Theming & Animation):**
  * Gerenciamento de **Dark Mode/Light Mode** via *CSS Custom Properties* (Variáveis de ambiente).
  * Animações fluidas de `@keyframes` (Efeito Ken Burns no Hero background, efeito de cascata/delay na galeria de fotos e simulação física de flutuação de texto).
* **JavaScript ES6+:** Manipulação assíncrona do DOM, tratamento de eventos mobile (*swipe gestures*) e gerenciamento de estados locais com `localStorage`.
* **Firebase Firestore:** Banco de dados NoSQL integrado de forma modular (`v10.8.0`) escutando alterações via sockets (`onSnapshot`) para atualização em tempo real do mural de recados sem necessidade de recarregar a página.
* **Leaflet.js API:** Motor de mapas geográficos customizado com injeção de marcadores HTML/SVG dinâmicos.
* **Chatbase API:** Integração assíncrona orientada a objetos (`window.chatbase`) de um agente de inteligência artificial generativa.
* **Intersection Observer API:** Monitoramento nativo do viewport para disparar animações e fazer o *lazy loading* de mídia pesada apenas quando necessário, otimizando o consumo de memória.
* **Acessibilidade (VLibras):** Widget oficial integrado para tradução simultânea em Libras, estendendo a inclusão do projeto.

## ⚡ Diferenciais Técnicos e Performance

* **Non-Blocking JS:** Scripts de terceiros (Chatbase, Firebase e VLibras) injetados de forma assíncrona para garantir que o *First Contentful Paint* (FCP) da página principal seja quase instantâneo.
* **Layout Fluido:** Design responsivo adaptado cirurgicamente desde telas mobile pequenas até monitores ultrawide através de funções CSS como `clamp()`.
* **Clean Code:** Organização lógica legível, modular e de fácil manutenção, mesmo concentrada em arquitetura limpa de arquivo estático.
